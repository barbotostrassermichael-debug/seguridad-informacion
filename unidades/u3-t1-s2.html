Introducción

La aritmética modular es la herramienta principal para desarrollar algoritmos de cifrado. En criptografía, trabajamos dentro de un cuerpo finito, donde las operaciones se realizan bajo un módulo n.

a mod n = b

Conceptos Fundamentales

1. Zona Canónica

Los restos de un número n siempre van desde 0 hasta n-1. Por ejemplo, en módulo 13, el resto 5 tiene una clase de equivalencia que incluye números como -21, -8, 18 y 31.

2. Inverso Aditivo

Es el número que, sumado a otro, da como resultado el elemento neutro (0) en el módulo correspondiente.

Ejemplo: En módulo 27, el inverso aditivo de 12 es 15.
Porque: 12 + 15 = 27 ≡ 0 (mod 27).

3. El operador XOR (OR Exclusivo)

Muy utilizado en cifrado de bits. Es una función involutiva, lo que significa que el inverso de un valor bajo XOR es el mismo valor. Es la base de algoritmos modernos como IDEA.

Inverso Multiplicativo

Es fundamental para la criptografía asimétrica. Para que un número a tenga inverso en módulo n, debe cumplirse que sean primos relativos:

mcd(a, n) = 1
Caso Práctico (Cifrado Afín):
Buscamos el inverso de 3 en módulo 10.
Multiplicamos 3 por todos los restos hasta encontrar el que da 1:
Operación3*03*13*23*33*43*53*63*73*83*9
Resultado mod 100369258147
Conclusión: El inverso de 3 en mod 10 es 7.

Nota: Para números grandes, se utiliza el Algoritmo Extendido de Euclides (AEE).
